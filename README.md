# Firefox
- Firefox nace como una alternativa a los navegadores propietarios, destacando en un inicio la ligereza y sus personalización misma que no ha cambiado. Su arquitectura priorizan el control del usuario sobre la interfaz y el comportamiento del navegador.
Al ser un navegador de código abierto, permite que personal ajeno al software, visualice, modifique, reduce y/o explote su funcionamiento. A diferencia de navegadores controlados por corporaciones con modelos publicitarios centrados en integración de servicios, Mozilla prioriza privacidad, interoperabilidad y control del usuario.
